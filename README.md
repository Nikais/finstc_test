# JSON-SCHEMA
## GET /api/car/
This method return list of cars  
Parameters: None  
Return: List of Cars

```json
[
    {
        "id": 6,
        "vin": "1234567890qwertyu",
        "brand": 1,
        "mark": "Corolla",
        "year": 1999,
        "price": 100000,
        "dealer": 2
    },
    ...
    {
        "id": 7,
        "vin": "1234567890qwertya",
        "brand": 1,
        "mark": "Camry",
        "year": 2007,
        "price": 2000000,
        "dealer": null
    }
]
```

## POST /api/car/\<id\>/
This method for create Car object  
Parameters: fields of Car  
Return: Car object
```json
{
    "id": 6,
    "vin": "1234567890qwertyu",
    "brand": 1,
    "mark": "Corolla",
    "year": 1999,
    "price": 100000,
    "dealer": 2
}
```

## PUT /api/car/\<id\>/
Parameters: one or all fields of Car  
Return: Car object
```json
{
    "id": 6,
    "vin": "1234567890qwertyu",
    "brand": 1,
    "mark": "Camry",
    "year": 1999,
    "price": 100000,
    "dealer": 2
}
```

## DELETE /api/car/\<id\>/
Delete Car object by ID  
Parameters: None
Return: None
